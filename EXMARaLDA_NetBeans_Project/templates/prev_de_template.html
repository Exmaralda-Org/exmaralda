Dies sind Previews (Vorabversionen) der nächsten offiziellen EXMARaLDA-Version. Sie enthalten Änderungen, Verbesserungen und Bugfixes für die letzte offizielle EXMARaLDA-Version.

Bitte beachten Sie: die aktuellen Previews für Windows und MAC OS sind neue Distributionen für 64bit Systeme / Java 11+ / OpenJDK. Die MAC-Versionen sind unter MAC OS 10.15 (Catalina) lauffähig. In diesen Versionen gibt es viele grundlegende Änderungen, u.a. an den Media-Playern. Die Versionen sind mehrfach getestet, können aber dennoch Fehler enthalten. Wir freuen uns über Testberichte an exmaralda-support AT uni-hamburg.de.

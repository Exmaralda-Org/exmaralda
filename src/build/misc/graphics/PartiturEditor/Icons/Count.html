General Toolbar Button Graphics

About

  Description: Provide information regarding the application.
  Graphics:  
  Name: About {Application Name}
  Tool Tip: About {Application Name}
  Shortcut:  
  Mnemonic: A, B, T
  File Name: .../toolbarButtonGraphics/general/About16.gif
    .../toolbarButtonGraphics/general/About24.gif
  Other Notes: For a more complete description of design considerations for the Java look and feel and "About", please see the "Designing About Boxes" section in the Java look and feel Design Guidelines.

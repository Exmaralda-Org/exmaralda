TEI Drop

TEI Drop is a tool supporting you in the conversion of spoken language transcriptions from one of several tool formats to an XML representation which conforms to the suggestions of the P5 version of Text Encoding Initiative's (TEI) Guidelines. TEI Drop is a droplet which means that you specify the input files by dragging them from your system's file browser (e.g. Windows Explorer or Macintosh Finder) onto the tool's user interface and dropping them there.


Input files

TEI Drop accepts and is able to process the following tool formats:

  • CHAT, the format read and written by the CLAN tool from the CHILDES project
  • EAF, the format read and written by the ELAN tool from the MPI in Nijmegen
  • EXB, the format read and written by the EXMARaLDA Partitur-Editor
  • FLK, the format read and written by the FOLK editor
  • TRS, the format read and written by the Transcriber editor
TEI Drop recognizes the input files by their file name extensions, so it expects input files to end in .cha, .eaf, .exb, .flk and .trs, respectively.


Parsing options

TEI Drop allows you to parse the document according to one of several transcription conventions which you specify in the drop down menu entitled Parse method At present, the conventions HIAT, cGAT and CHAT are supported. In order to leave the document unparsed, simply specify the parsing method as None.


Output options

The ouput file written by TEI Drop will have the same name as the input file with the original suffix (.cha, .eaf, etc.) replaced by the suffix you specify in the field suffix for ouput. If you choose Write output to the same directory, the output file will be written to the same directory as the input file. If you choose ... to a separate directory, all files will be written to the directory given in the text field. You can browse for a directory using the Browse... button.
